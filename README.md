You will need to clone the following repositories.

* [Owlvey Api](https://github.com/owlvey/owlvey_api)

    scripts\migrations.bash
    ```
    ```
    scripts\run_server.bash
    ```
* [Owlvey Authority](https://github.com/owlvey/owlvey_identity)

    scripts\migrations.bash
    ```
